# Airline Reservation System (Java)
This is a console-based airline reservation system developed using Java.

## Features
- Add flights
- View available flights
- Book seats
- Save flight data using File Handling

## Concepts Used
- Object-Oriented Programming (OOP)
- Arrays
- File I/O
- Serialization

## How to Run
1. Compile the program:
   javac AirlineReservationSystem.java

2. Run the program:
   java AirlineReservationSystem
